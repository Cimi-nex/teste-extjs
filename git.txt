para ver o status do meu cod
    git status

para ver oq eu mudei
    git diff

para add tudo que eu mudei 
    git add .

para escrever uma mensagem que eu mudei
    git commit -m "mensagem aqui"

para subir o cod que mudei para o github
    git push
 
para trocar branch 
    git switch (nome da branch)

para criar uma branch
    git switch -c (nome da branch)

para sincronizar meu cod local com o cod do github
    git pull

para voltar para o commit antigo 
    git restore .
